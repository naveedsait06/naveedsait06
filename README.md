## Projects

### 🔍 A11y Auditor Pro
**[Live Demo](https://naveedsait06.github.io/a11y-auditor-pro/)** · Vanilla JS · May 2026

A client-side web accessibility auditing tool that programmatically detects WCAG compliance failures — missing ARIA labels, alt attributes, and more.

- Built a custom virtual DOM parser using `DOMParser()` to safely traverse and analyze node trees
- Generates real-time remediation snippets so developers know exactly what to fix
- Uses the JavaScript Blob API to export downloadable `.txt` audit reports
- Fully mobile-responsive with a Glassmorphism UI, CSS grid/flexbox layout, and a custom Dark Mode toggle

> This project came from a genuine interest in inclusive web design. Accessibility tooling is still underbuilt — I wanted to understand what it actually takes to detect failures at the DOM level.
